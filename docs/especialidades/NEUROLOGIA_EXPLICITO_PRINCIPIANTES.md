## PARTE 1: ¿QUÉ ES EL CEREBRO?

### El Órgano Más Complejo
Tu cerebro es:
- Órgano más importante (controla TODO)
- Peso: ~1.4 kg (3 libras)
- Neuronas: ~86 BILLONES
- Conexiones: ~100 TRILLONES

### Las 4 Regiones Principales
```
         FRONTAL
    (Decisiones, Movimiento)
           │
TEMPORAL   │   PARIETAL
(Memoria) │   (Sensación)
           │
         OCCIPITAL
        (Visión)
```

| Región | Función |
|--------|---------|
| **Frontal** | Decisiones, movimiento voluntario, personalidad |
| **Temporal** | Memoria, audición, emoción |
| **Parietal** | Sensación, coordinación, espacio |
| **Occipital** | Visión y procesamiento visual |

## PARTE 2: ¿QUÉ ES EL EEG?

### Definición Simple
El EEG mide la **actividad eléctrica del cerebro** con electrodos en el cuero cabelludo.

### ¿POR QUÉ EL CEREBRO ES ELÉCTRICO?
```
Neurona = Célula nerviosa
│
├─ Soma: Cuerpo celular
├─ Dendritas: Antenas receptoras
└─ Axón: Cable transmisor

Cuando piensas:
1. Las neuronas se comunican
2. Usan iones (moléculas cargadas)
3. Crean voltajes eléctricos pequeños
4. Electrodo EEG lo detecta
```

### CANALES EEG (Dónde Medimos)

```
       ┌─ Fp1    Fp2 ─┐  FRONTAL
       │                │ (Decisión, movimiento)
       │ F3   Fz   F4  │
       │                │
       ├─ T3 ─ C3 ─ T4 ─┤ CENTRAL
       │    (Sensación)  │
       ├─ T5   P3  T6 ─┤ PARIETAL
       │    (Espacio)   │
       └─ O1 ─ Oz ─ O2 ─┘ OCCIPITAL
              (Visión)
```

**Sistema 10-20:** Electrodo cada 10% y 20% de la distancia entre puntos de la cabeza

## PARTE 3: BANDAS DE FRECUENCIA EEG

### ¿QUÉ SON BANDAS?
El cerebro produce diferentes "frecuencias" según qué hagas:

```
1 segundo = 1000 milisegundos

Ondas rápidas = Pensamiento rápido
Ondas lentas = Descanso/sueño
```

### LAS 5 BANDAS PRINCIPALES

#### 1. DELTA (0.5-4 Hz) - Sueño Profundo
```
Onda lenta y profunda:
     ┌─────┐     ┌─────┐
     │     │     │     │
─────┘     └─────┘     └─────
  
0.5-4 ciclos por segundo

QUÉ SIGNIFICA:
✓ Sueño profundo (Etapa N3)
✓ Restauración física
✓ Consolidación de memoria
✓ Liberación de hormona del crecimiento

ANORMAL si:
- Despierto con mucho delta = Demencia
- Despierto con mucho delta = Trauma cerebral
```

#### 2. THETA (4-8 Hz) - Somnolencia
```
Onda lenta moderada:
    ┌───┐   ┌───┐   ┌───┐
    │   │   │   │   │   │
────┘   └───┘   └───┘   └───

4-8 ciclos por segundo

QUÉ SIGNIFICA:
✓ Transición vigilia-sueño
✓ Sueño ligero (Etapa N1-N2)
✓ Meditación profunda
✓ Estrés moderado

ANORMAL si:
- Mucho theta despierto = Deficiencia de atención
- Patrón theta anormal = Epilepsia
```

#### 3. ALPHA (8-12 Hz) - Relajación Despierta
```
Onda moderada regular:
   ┌──┐   ┌──┐   ┌──┐
   │  │   │  │   │  │
───┘  └───┘  └───┘  └───

8-12 ciclos por segundo

QUÉ SIGNIFICA:
✓ Despierto relajado
✓ Ojos cerrados en reposo
✓ Meditación
✓ Paz y calma
✓ Mejor en región occipital (visión)

ANORMAL si:
- Poco alpha = Ansiedad, estrés
- Mucho alpha despierto = Depresión
```

#### 4. BETA (12-30 Hz) - Alerta y Concentración
```
Onda rápida:
  ┌─┐  ┌─┐  ┌─┐  ┌─┐
  │ │  │ │  │ │  │ │
──┘ └──┘ └──┘ └──┘ └──

12-30 ciclos por segundo

QUÉ SIGNIFICA:
✓ Despierto alerta
✓ Concentración
✓ Resolución de problemas
✓ Actividad cognitiva
✓ Normal durante el día

ANORMAL si:
- Mucho beta = Estrés crónico, ansiedad
- Anormal en sueño = Insomnio
```

#### 5. GAMMA (30-100 Hz) - Procesamiento Cognitivo
```
Onda muy rápida:
 ┌┐ ┌┐ ┌┐ ┌┐ ┌┐
 ││ ││ ││ ││ ││
─┘└─┘└─┘└─┘└─┘└─

30-100 ciclos por segundo

QUÉ SIGNIFICA:
✓ Procesamiento de información compleja
✓ Aprendizaje activo
✓ Memoria de trabajo
✓ Conciencia integrada

RARO si:
- Poco gamma = Pérdida cognitiva
- Gamma desorganizado = Esquizofrenia, autismo
```

## PARTE 5: EPILEPSIA - CUANDO EL CEREBRO "CORTOCIRCUITA"

### ¿QUÉ ES EPILEPSIA?
Descargas eléctricas anormales en el cerebro que causan crisis.

### PATRONES EEG DE EPILEPSIA

#### Patrón Punta-Onda (Spike-Wave)
```
Normal:         Epilepsia:
     ↓               ↓↓↓↓↓
    ┌┐            ┌┐┌┐┌┐
────┘└──          ┘└┘└┘└──── ← PUNTA rápida
                    (muy abrupta)

Significado:
⚠️ Neurona disparándose sin control
⚠️ Puede causar convulsión
⚠️ Requiere medicamento (anticonvulsivo)
```

### TIPOS DE CRISIS

| Tipo | EEG | Síntoma |
|------|-----|---------|
| **Ausencia** | Punta-onda 3/seg | Desconexión 5-10 seg |
| **Tónico-clónica** | Punta actividad rápida | Convulsión, pérdida conciencia |
| **Focal** | Punta en una región | Espasmos locales |
| **Compleja** | Punta en región temporal | Comportamiento confuso |

### ANTES/DURANTE/DESPUÉS DE CRISIS

```
ANTES (Aura):
- Hormigueo
- Olor extraño
- Miedo
- Cambios en EEG

DURANTE:
- Pérdida de conciencia
- Contracciones musculares
- EEG: Patrón anormal masivo

DESPUÉS (Período posictal):
- Confusión
- Sueño profundo
- Fatiga
- Dolor muscular
```

## PARTE 6: TRASTORNOS DEL SUEÑO

### INSOMNIO
```
DEFINICIÓN: No poder dormir bien
CAUSAS: Estrés, ansiedad, ruido, luz, medicamentos
EEG MUESTRA: Beta alto incluso en sueño
TRATAMIENTO: Higiene del sueño, medicación, CBT
```

### APNEA DEL SUEÑO
```
DEFINICIÓN: Pausas en la respiración durante el sueño
RIESGO: Baja oxígeno en cerebro → Muerte
EEG MUESTRA: Despertares frecuentes (micro-despertares)
TRATAMIENTO: CPAP mask
```

### NARCOLEPSIA
```
DEFINICIÓN: Ataques de sueño irresistibles
CAUSA: Falta de neuroinhibidor (hipocretina)
EEG MUESTRA: REM al inicio del sueño (anormal)
SÍNTOMA: Cataplexia (pérdida muscular despierto)
```

## PARTE 7: CÓMO LA IA DETECTA AUTOMÁTICAMENTE

### EL FLUJO AUTOMÁTICO

```
1. EEG entra
   ↓
2. IA extrae bandas (Delta, Theta, Alpha, Beta, Gamma)
   ↓
3. IA compara con patrones normales
   ↓
4. ¿Es normal? ¿Epilepsia? ¿Anomalía?
   ↓
5. Predicción: ¿Riesgo de crisis en próximas 24 horas?
   ↓
6. Digital Twin: Visualiza cerebro simulado
   ↓
7. Alertas: Si hay riesgo → Alerta inmediata
   ↓
8. Reporte: Documento automático
```

### CARACTERÍSTICAS QUE DETECTA IA

#### Espiga (Spike)
```
Onda muy abrupta:
     ↓
    ╱╲
───╱  ╲───
   <20ms

IA dice: "Posible epilepsia"
Confianza: 82%
```

#### Punta-Onda
```
Punta seguida de onda:
  ╱╲   ┐
 ╱  ╲ ╱ ╲
╱    ╱   ╲
      └──Onda

IA dice: "Probable ausencia (epilepsia menor)"
Confianza: 88%
```

#### Lentificación
```
Bandas anormalmente lentas:
     ┌─────┐
─────┘     └───── ← Delta excesivo despierto

IA dice: "Posible encefalopatía"
Confianza: 75%
```

## PARTE 9: CUÁNDO LLAMAR AL MÉDICO

### EMERGENCIA (Llamar 911)
- Convulsión activa
- Pérdida de conciencia
- Dolor de cabeza severo + fiebre + rigidez de nuca (meningitis)
- Accidente cerebrovascular (FACE, ARMS, SPEECH test)

### CONSULTA URGENTE
- Primer episodio convulsivo
- Cambio en patrón de convulsiones
- Dolor de cabeza persistente nuevo
- Mareos/vértigo severo

### CONTROL REGULAR
- Epilepsia: Cada 3-6 meses
- Tras medicación: Después de 1-2 semanas
- Anual si no hay problemas

## PARTE 10: PREGUNTAS FRECUENTES

### P: ¿Qué es una pequeña punta en EEG?
**R:** Puede ser normal (especialmente en sueño) o anormal.
- Contexto importa: ¿Dónde? ¿Cuándo? ¿Cuántas?

### P: ¿Puedo tener epilepsia sin espiga?
**R:** Sí. 20% de pacientes no muestran espigas.
Requiere varios EEGs o video-EEG.

### P: ¿Mi cerebro se "gasta"?
**R:** No se gasta como un motor.
Pero puede dañarse por:
- Lesión traumática
- Infarto cerebral
- Infección (meningitis)
- Degeneración (Alzheimer)

### P: ¿Es hereditaria la epilepsia?
**R:** Riesgo aumentado si familia tiene,
pero NO garantizado.
- Gen riesgo: 5-10%
- Factores ambientales importan
