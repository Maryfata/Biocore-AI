## INICIO RÁPIDO (5 MINUTOS)

### Paso 1: Selecciona tu Especialidad

```
¿Qué parte del cuerpo quieres medir?

🫀 CARDIOLOGÍA (Corazón)
   ├─ ¿Quiero medir mi corazón?
   ├─ ¿Tengo palpitaciones?
   ├─ ¿Tengo presión alta?
   └─ Selecciona → Cardiología

🧠 NEUROLOGÍA (Cerebro)
   ├─ ¿Quiero medir mis ondas cerebrales?
   ├─ ¿Tengo problemas de sueño?
   ├─ ¿Tengo epilepsia?
   └─ Selecciona → Neurología

💪 MUSCULOESQUELÉTICO (Músculos)
   ├─ ¿Quiero medir fatiga muscular?
   ├─ ¿Tengo debilidad?
   ├─ ¿Estoy en rehabilitación?
   └─ Selecciona → Musculoesquelético
```

### Paso 2: Toma la Medición

```
Una vez seleccionaste especialidad:

CARDIOLOGÍA:
├─ Coloca 3 electrodos en el pecho
├─ Presiona "Iniciar ECG"
├─ Espera 10 segundos
└─ El sistema analiza automáticamente

NEUROLOGÍA:
├─ Colócate el gorro EEG (si disponible)
├─ O usa auriculares Muse/Emotiv
├─ Presiona "Iniciar medición EEG"
├─ Espera 30 segundos (mínimo)
└─ El sistema analiza automáticamente

MUSCULOESQUELÉTICO:
├─ Coloca EMG sensor en el músculo
├─ Presiona "Iniciar medición EMG"
├─ Contrae músculo según instrucciones
├─ Sistema mide 10-30 segundos
└─ El sistema analiza automáticamente
```

### Paso 3: Recibe Resultados Automáticos

```
TODO sucede en < 5 segundos:

1️⃣ ANÁLISIS AUTOMÁTICO
   ✓ Sistema analiza tu medición
   ✓ Compara con patrones normales
   ✓ Detecta anomalías

2️⃣ IA AUTOMÁTICA
   ✓ Predice riesgos futuros
   ✓ Sugiere tratamientos
   ✓ Genera recomendaciones

3️⃣ DIGITAL TWIN
   ✓ Visualiza tu corazón/cerebro/músculos
   ✓ Muestra estado actual
   ✓ Predice evolución

4️⃣ ALERTAS
   🟢 Verde = Todo normal
   🟡 Amarillo = Vigilancia necesaria
   🔴 Rojo = Consulta médica
   🔴🔴 Rojo crítico = EMERGENCIA

5️⃣ REPORTE AUTOMÁTICO
   ✓ Documento listo para tu médico
   ✓ Explicado para principiantes
   ✓ Incluye recomendaciones
```

## EXPLICACIONES DETALLADAS

### Para Cardiología

Cada resultado automático incluye:

```
1. ¿QUÉ SIGNIFICA?
   Definición simple de lo que encontramos

2. ¿POR QUÉ IMPORTA?
   Por qué este hallazgo es relevante
   para tu salud

3. ¿CUÁL ES EL RANGO NORMAL?
   Qué se considera normal/anormal

4. ¿QUÉ HACER AHORA?
   Acciones concretas recomendadas

5. ¿CUÁNDO CONSULTAR MÉDICO?
   Signos de alerta

6. EXPLICACIÓN DE IA
   Por qué la IA llegó a esta conclusión
   (SHAP/LIME explicability)
```

### Para Neurología

```
Similar pero con foco en:
- Bandas de frecuencia explicadas
- Correlación con síntomas
- Cambios normales vs patológicos
- Estadios de sueño
- Riesgo de epilepsia
```

### Para Musculoesquelético

```
Enfoque en:
- Fatiga muscular explicada
- Patrones de activación
- Comparación con normal
- Pronóstico de recuperación
- Recomendaciones de ejercicio
```

## PREGUNTAS FRECUENTES

### P: ¿Qué tan precisa es la IA?
**R:** Confianza típica: 85-92%
- Depende de calidad de señal
- Validada contra estándares médicos
- Se mejora con más datos

### P: ¿Puedo confiar en el diagnóstico?
**R:** La IA es herramienta, no diagnóstico final
- Siempre consulta médico
- IA ayuda a filtrar y priorizar
- Médico toma decisión final

### P: ¿Cuándo debo repetir mediciones?
**R:** Depende de tu condición
- Cardiología: Semanal si con medicamento
- Neurología: Diariamente si epiléptico
- Musculoesquelético: 2-3× por semana en rehab

### P: ¿Mis datos están seguros?
**R:** Sí
- Encriptación de datos
- Cumplimiento HIPAA/GDPR
- Acceso controlado
- Logs de auditoría

### P: ¿Funciona sin internet?
**R:** Parcialmente
- Análisis locales: Sí
- IA más complejas: Requiere conexión
- Sincronización cuando hay conexión
