## PARTE 1: ¿QUÉ ES EL CORAZÓN?

### El Órgano Más Importante
Tu corazón es una **bomba muscular** del tamaño de tu puño que:
- Bombea sangre a todo tu cuerpo
- Funciona **sin parar 24/7** (~100,000 latidos/día)
- Es controlado automáticamente (no necesitas pensar en él)

### Las Cuatro Cámaras
```
        ┌─────────┐
        │ Aurícula│  ← Recibe sangre
        │Derecha  │
        └────┬────┘
             │ (Válvula Tricúspide)
        ┌────▼────┐
        │Ventrículo│ ← Bombea sangre a los pulmones
        │Derecho   │
        └─────────┘
        
        ┌─────────┐
        │ Aurícula│  ← Recibe sangre de pulmones
        │Izquierda│
        └────┬────┘
             │ (Válvula Mitral)
        ┌────▼────┐
        │Ventrículo│ ← Bombea sangre a TODO el cuerpo
        │Izquierdo │
        └─────────┘
```

**Función**: 
- Sangre sucia (poco oxígeno) entra en aurícula derecha
- Va a pulmones a conseguir oxígeno
- Vuelve con oxígeno a aurícula izquierda
- Se bombea a TODO el cuerpo

## PARTE 2: ¿QUÉ ES EL ECG?

### Definición Simple
El ECG es como una **película de la actividad eléctrica** del corazón.

### ¿Por Qué el Corazón es Eléctrico?
Cada célula del corazón tiene **iones** (moléculas con carga eléctrica).
Cuando el corazón se contrae:
1. Los iones se mueven
2. Crean pequeños voltajes eléctricos
3. Estos se pueden medir con electrodos en la piel

### Las Partes de una Onda ECG

```
         ┌─ P
         │  Onda P: Aurícula se contrae (despolarización auricular)
         │
     ────┴─────────────┐
     QRS complex      │ Complejo QRS: Ventrículo se contrae
                      │ (despolarización ventricular)
                      │ La parte más importante
                      │
                      └──────┐
                             │ Onda T: Ventrículo se relaja
                             │ (repolarización ventricular)
```

### ¿Qué Significa Cada Parte?

| Componente | Qué Hace | Duración Normal |
|-----------|----------|-----------------|
| **Onda P** | Aurícula se contrae y empuja sangre | 0.06-0.12 segundos |
| **Intervalo PR** | Espera a que ventrículo se llene | 0.12-0.20 segundos |
| **Complejo QRS** | **VENTRÍCULO SE CONTRAE** (lo más importante) | 0.08-0.12 segundos |
| **Intervalo ST** | Ventrículo contraído, esperando relajarse | Línea recta |
| **Onda T** | Ventrículo se relaja | Normal si hacia arriba |

### Ritmo Cardíaco Normal
```
P─┐    QRS    ┌─T
  └────┘      └─
|──────────────────| = 1 segundo

En 1 segundo = 1 latido
En 60 segundos = 60 latidos/minuto = NORMAL
```

## PARTE 3: ARRITMIAS - CUANDO EL CORAZÓN FALLA

### ¿QUÉ ES UNA ARRITMIA?
Es cuando el corazón late de forma **ANORMAL** (demasiado rápido, demasiado lento, o irregular).

### TIPOS PRINCIPALES

#### 1. RITMO SINUSAL NORMAL (NSR)
```
P─┐    QRS    ┌─T     P─┐    QRS    ┌─T
  └────┘      └─        └────┘      └─
  |─ NORMAL ─|─ NORMAL ─|

✓ Frecuencia: 60-100 BPM
✓ Regular y rítmico
✓ Significa: Tu corazón está sano
```

#### 2. FIBRILACIÓN AURICULAR (FA)
```
Ruido───P─────QRS───┌─T────────ruido────QRS
electrod confuso   │
        │          └─ (No hay patrón claro)

⚠️ Frecuencia: 100-160 BPM (muy variable)
⚠️ Irregular (latidos no sincronizados)
⚠️ Significa: Las aurículas tiemblan sin contraerse bien
⚠️ Riesgo: Coágulos de sangre → ICTUS
```

#### 3. BRADICARDIA (Corazón Lento)
```
P─┐    QRS      ┌─T
  └────────┘    └────────
  |────LENTO─────| = Más de 1 segundo

⚠️ Frecuencia: < 60 BPM
⚠️ Demasiado lento
⚠️ Significado: No bombea suficiente sangre
⚠️ Causas: Edad, medicamentos, problemas eléctricos
```

#### 4. TAQUICARDIA (Corazón Rápido)
```
P QRS  ┌T
 └┘└───┘
|RÁPIDO| = Menos de 1 segundo

⚠️ Frecuencia: > 100 BPM
⚠️ Demasiado rápido
⚠️ Significado: Fuerza innecesaria en el corazón
⚠️ Causas: Estrés, cafeína, fiebre, ejercicio
```

#### 5. BLOQUEO AV (Corazón Desconectado)
```
P─┐        P─┐        P─┐
  │        │          │
  │ (Espera larga)    │
  │        │          │
  └─QRS    │ (No hay QRS)
           └─(Se perdió)

⚠️ Desconexión entre aurículas y ventrículos
⚠️ Grados:
   - Grado 1: Solo espera larga
   - Grado 2: A veces pierde contracciones
   - Grado 3: Conexión completamente perdida
⚠️ Significado: Aurículas y ventrículos no coordinan
```

## PARTE 4: HRV - VARIABILIDAD DEL RITMO CARDÍACO

### ¿QUÉ ES HRV?
Es la **variación en el tiempo entre latidos**.

### LÓGICA SIMPLE
```
Persona RELAJADA:
80 ms - 95 ms - 82 ms - 100 ms - 85 ms
     ↑              ↑              ↑
   Variación = ALTA = BUENA SALUD

Persona ESTRESADA:
80 ms - 79 ms - 81 ms - 80 ms - 79 ms
     ↑                            ↑
   Variación = BAJA = ESTRÉS
```

### ¿POR QUÉ IMPORTA?
Un HRV alto significa:
- ✓ Tu sistema nervioso es flexible
- ✓ Puedes cambiar de ritmo fácilmente
- ✓ Menor riesgo de ataque cardíaco
- ✓ Mejor recuperación del estrés

Un HRV bajo significa:
- ⚠️ Tu sistema nervioso está "rígido"
- ⚠️ Dificultad para adaptarse al cambio
- ⚠️ Mayor estrés
- ⚠️ Mayor riesgo de problemas cardíacos

### MEDIDAS DE HRV

#### 1. SDNN (Desviación Estándar)
- Qué mide: Variación total entre todos los latidos
- Rango normal: 30-100 ms
- Si bajo: Estrés o enfermedad

#### 2. RMSSD (Raíz Cuadrada del Promedio)
- Qué mide: Cambios rápidos de latido a latido
- Refleja: Actividad del sistema parasimpático (relajación)
- Rango normal: 20-100 ms
- Si alto: Más relajado

#### 3. LF/HF (Balance Simpático/Parasimpático)
```
Tu sistema nervioso tiene dos "velocidades":

SIMPÁTICO = Acelerador (estrés, lucha)
├─ Aumenta: Frecuencia cardíaca, presión sanguínea
└─ Aparece en: Peligro, estrés, ejercicio

PARASIMPÁTICO = Freno (relajación, reposo)
├─ Aumenta: Digestión, calma, sueño
└─ Aparece en: Meditación, descanso, sleep

LF/HF = Acelerador / Freno
- Ratio alto (> 2.5): Estrés dominante
- Ratio bajo (< 1.5): Relajación dominante
```

## PARTE 5: PRESIÓN ARTERIAL

### ¿QUÉ ES?
Es la **fuerza con la que la sangre presiona las arterias**.

### CÓMO SE MIDE
```
SISTÓLICA / DIASTÓLICA
   ↑            ↑
Ventrículo   Ventrículo
se contrae   se relaja

Ejemplo: 120 / 80 mmHg
```

### CATEGORÍAS

| Categoría | Sistólica | Diastólica | Significado |
|-----------|-----------|-----------|------------|
| **Normal** | < 120 | < 80 | ✓ Salud óptima |
| **Elevada** | 120-129 | < 80 | ⚠️ Vigilancia |
| **Etapa 1** | 130-139 | 80-89 | ⚠️ Medicamento posible |
| **Etapa 2** | ≥ 140 | ≥ 90 | ⚠️ Medicamento urgente |
| **Crisis** | > 180 | > 120 | 🚨 EMERGENCIA |

### POR QUÉ IMPORTA
```
Presión ALTA:
- Daña arterias (endurecimiento)
- Fuerza excesiva en corazón
- Riesgo de infarto, ictus
- Síntomas: Dolor de cabeza, visión borrosa

Presión BAJA:
- Poca oxigenación de órganos
- Mareos, desmayos
- Daño renal
- Síntomas: Fatiga, debilidad
```

## PARTE 6: CÓMO NUESTRO SISTEMA DETECTA AUTOMÁTICAMENTE

### EL FLUJO AUTOMÁTICO

```
1. MEDICIÓN
   Electrodos en pecho detectan electricidad
   ↓
2. SEÑAL DIGITAL
   Se convierte a números que la IA puede analizar
   ↓
3. IA AUTOMÁTICA
   Analiza QRS, P, T, duración, ritmo
   ↓
4. DETECCIÓN AUTOMÁTICA
   ¿Es normal? ¿Arritmia? ¿Bloqueo?
   ↓
5. PREDICCIÓN AUTOMÁTICA
   ¿Qué pasa en 30 días? ¿Riesgo?
   ↓
6. DIGITAL TWIN
   Visualiza corazón simulado
   ↓
7. ALERTAS
   Si hay problema → Alerta inmediata
   ↓
8. REPORTE
   Documento automático para médico
```

### ¿QUÉ HACE LA IA?

#### Reconoce Patrones
```
Entrada: Onda ECG
         |
         v
¿Es esto...?  ← IA compara con millones
         |
         ├─ Normal (95% confianza)
         ├─ FA (87% confianza)
         ├─ Bradicardia (92% confianza)
         └─ Bloqueo AV (80% confianza)
```

#### Explica por Qué
```
"Detecté FA porque:"
1. RR intervals muy variables (variación >30%)
2. No hay ondas P claras
3. Complejo QRS irregular
4. Frecuencia > 100 BPM
→ Confianza: 87%
```

#### Predice el Futuro
```
"Basado en tu FA, en 30 días:"
├─ Riesgo de ictus: 2.3%
├─ Necesidad de medicamento: Alta
└─ Pronóstico con tratamiento: Bueno
```

## PARTE 7: ¿QUÉ SIGNIFICA CADA ALERTA?

### 🟢 VERDE (Normal)
- Ritmo sinusal normal
- Frecuencia 60-100 BPM
- Presión 120/80 o menor
- **Acción**: Continúa monitoreo regular

### 🟡 AMARILLO (Vigilancia)
- Ritmo irregular ocasional
- Frecuencia 50-60 o 100-110 BPM
- Presión 130-140 / 80-90
- **Acción**: Aumenta mediciones, evita estrés

### 🔴 ROJO (Atención)
- Arritmia detectada
- Frecuencia < 50 o > 120 BPM
- Presión > 140/90
- **Acción**: Consultar médico hoy

### 🔴🔴 ROJO CRÍTICO (EMERGENCIA)
- Fibrilación auricular rápida
- Frecuencia > 160 BPM
- Presión > 180/120
- Dolor en el pecho
- **Acción**: 🚑 LLAMAR AMBULANCIA

## PARTE 8: CÓMO MANTENER UN CORAZÓN SANO

### DIETA PARA EL CORAZÓN
```
COME MÁS:
├─ Frutas (manzana, plátano)
├─ Verduras (brócoli, espinaca)
├─ Pescado graso (salmón)
├─ Nueces
└─ Avena

EVITA:
├─ Sal excesiva
├─ Grasas saturadas
├─ Azúcares refinados
├─ Alcohol excesivo
└─ Comida ultra procesada
```

### EJERCICIO
```
OBJETIVO: 150 minutos moderados/semana

EJEMPLOS:
- Caminar rápido: 30 min × 5 días = 150 min ✓
- Correr: 20 min × 3 días = 60 min + 
  + caminar: 30 min × 3 días = 90 min = 150 min ✓
- Natación: 25 min × 6 días = 150 min ✓

BENEFICIOS:
✓ Fortalece el corazón
✓ Baja presión
✓ Mejora HRV
✓ Aumenta energía
```

### ESTRÉS Y SUEÑO
```
ESTRÉS MALO:
├─ Sube presión
├─ Baja HRV
├─ Arritmias
└─ Inflamación

MANEJO:
- Meditación: 10 min/día
- Respiración profunda (4-7-8)
- Hobby relajante
- Conexión social

SUEÑO:
├─ 7-9 horas TODAS las noches
├─ Horario consistente
├─ Cuarto oscuro y frío
└─ Sin pantallas 1 hora antes
```

## PARTE 9: PREGUNTAS FRECUENTES

### P: ¿Qué es la frecuencia cardíaca normal?
**R:** 60-100 BPM (latidos por minuto) en reposo.
- Atletas: 40-60 BPM (más eficientes)
- Niños: 70-100 BPM (natural)

### P: ¿Es peligrosa la FA?
**R:** Sí, pero controlable. Riesgos:
- Ictus (5-7 × más riesgo)
- Insuficiencia cardíaca
- Complicaciones
**Solución:** Medicamentos + anticoagulantes

### P: ¿Puedo vivir con arritmia?
**R:** Sí, muchas personas. Depende:
- Tipo de arritmia
- Frecuencia
- Síntomas
- Tratamiento

Muchas personas con FA viven 20+ años con tratamiento.

### P: ¿Mi corazón se "cansa"?
**R:** No se cansa como músculo esquelético.
**Pero puede debilitarse por:**
- Presión alta crónica
- Infarto previo
- Diabetes
- Enfermedades genéticas

### P: ¿Es malo saltarse un latido?
**R:** Ocasionalmente = NORMAL.
- El 50% de las personas las tienen
- Más comunes con cafeína, estrés, falta de sueño
- Preocupante si: Mucha frecuencia + mareos + dolor

## PARTE 10: RESUMEN RÁPIDO

```
CORAZÓN = Bomba de sangre de 4 cámaras

ECG = Película de la electricidad del corazón
      ├─ P: Aurícula se contrae
      ├─ QRS: Ventrículo se contrae (IMPORTANTE)
      └─ T: Ventrículo se relaja

RITMO NORMAL = P → QRS → T repetido 60-100× por minuto

ARRITMIAS = Cuando el patrón falla
            ├─ FA: Irregular y rápido (peligro)
            ├─ Bradicardia: Muy lento
            ├─ Taquicardia: Muy rápido
            └─ Bloqueo: Sin conexión

HRV = Variación entre latidos (alto = sano)

PRESIÓN = Fuerza de sangre en arterias
          ├─ Normal: 120/80
          └─ Alto: > 140/90 (peligro)

IA AUTOMÁTICA = Analiza, predice, alerta en milisegundos
```

## PARTE 11: CUÁNDO LLAMAR AL MÉDICO

### LLAMA HOY (No esperes)
- Dolor en el pecho
- Falta de aire
- Mareos graves
- Desmayos
- Palpitaciones con dolor

### LLAMA ESTA SEMANA
- Palpitaciones frecuentes
- Presión alta persistente
- Inflamación de piernas
- Cansancio anormal

### CONTROL REGULAR
- Cada año si normal
- Cada 3 meses si con medicamento
- Mensualmente si cambios recientes
